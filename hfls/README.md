# HFLS - 소방시설 연혁 법령 적용 지원 시스템

사용승인 시점별 소방법령 자동 판별 및 의무 소방시설 조회 시스템

---

## 로컬 실행 방법

```bash
# 1. Node.js 설치 확인 (https://nodejs.org)
node -v

# 2. 패키지 설치
npm install

# 3. 실행
npm start
# → http://localhost:3000 에서 접속 가능
```

---

## Vercel 무료 배포 방법 (권장)

### 1단계: GitHub에 코드 올리기
1. https://github.com 회원가입 (무료)
2. 새 Repository 생성 → 이름: `hfls`
3. 이 폴더의 파일을 전부 업로드

### 2단계: Vercel 배포
1. https://vercel.com 접속 → GitHub 계정으로 로그인
2. "New Project" → GitHub에서 `hfls` repository 선택
3. 설정 없이 "Deploy" 클릭
4. 배포 완료 후 링크 생성 (예: `hfls.vercel.app`)

→ 이 링크를 누구와도 공유 가능. 스마트폰에서도 접속 가능.

---

## Netlify 무료 배포 방법 (대안)

1. `npm run build` 실행 → `build` 폴더 생성
2. https://netlify.com 접속 → 로그인
3. `build` 폴더를 Netlify 화면에 드래그 앤 드롭
4. 즉시 배포 완료

## 기능

- 사용승인일 입력 → 적용 법령 시대 자동 판별 (4단계)
- 용도·면적·층수 기반 의무 소방시설 자동 조회
- 현행 기준(NFTC) 대비 추가 시설 비교
- 증축 특례 이중기준 자동 분리 처리 (시행령 제15조의2)
- 법령 변천 이력 전체 조회
